Detailed Description

Klasa odpowiedzialna za komunikację z API GIOŚ (Główny Inspektorat Ochrony Środowiska)

Wszystkie metody fetch() pobierają dane z internetu i zapisują je do plików JSON w folderze API/. W przypadku braku połączenia używają danych zapisanych podczas ostatniej sesji online.

Przykład użycia: ApiClient api; api.fetchStations(); api.fetchSensors(52); api.fetchMeasurements(295);
